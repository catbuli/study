# 深浅拷贝

## 介绍

浅拷贝（shallow copy）：只复制指向某个对象的指针，而不复制对象本身，新旧对象共享一块内存；

深拷贝（deep copy）：复制并创建一个一摸一样的对象，不共享内存，修改新对象，旧对象保持不变

### 浅拷贝

```javascript
var a = {name:"a",age:15};
var b = a;

b.age = 20;

console.log(a);
console.log(b);
```

我们将a的值传递给了b，因为js默认浅拷贝，所以其实我们是将a的地址传递给了b。

所以我们在修改b的age时，a的age也一同更改了，因为这两个变量指向的是同一块内存空间。
